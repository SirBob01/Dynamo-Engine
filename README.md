# Dynamo Engine

A 2D game engine powered by SDL2.

Features:
- Layered Scene Stack
- Input binding
- Animated sprites
- Texture manager
- Audio streaming (separate channels for music, ambience, and sound effects)
- Rasterizing primitives
- Custom networking protocol

# Dependencies

This engine wraps around SDL2 and its extensions:
- SDL2_image
- SDL2_mixer
- SDL2_ttf
- SDL2_net
